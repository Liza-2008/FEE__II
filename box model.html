What the Box Model is

Every HTML element is represented as a rectangular box made up of (from outside to inside):
  1. Margin — transparent space outside the border; separates elements.
  2. Border — sits between margin and padding; draws an edge around the element.
  3. Padding — space between the border and the content; increases the clickable area and spacing inside the box.
  4. Content — the width/height set on the element (text, images, etc.).

Example code

<div class="box" style="width:200px; padding:20px; border:6px solid #1e88e5; margin:24px;">CONTENT</div>

Note about box-sizing: content-box (the default) means the width and height apply to the content box only; padding and border add to the outside size. With border-box, the width includes padding and border (often easier for layouts).

Quick reference

  • To reset layouts: *{box-sizing:border-box} is a common pattern.
  • Margins between block elements collapse vertically — two stacked margins may overlap.
  • Padding never collapses with margins.
